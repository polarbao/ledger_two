# LedgerTwo Chat Save

Read first:

1. `AGENTS.md`

## Rules

- Do not save secrets, credentials, cookies, tokens, or unrelated personal data.
- Do not invent validation results.
- Keep the archive concise and reusable.
- Include decisions, constraints, files touched, commands run, verification status, risks, and next suggested prompt.

## Output

- Saved path
- Summary of what was saved
- Any omitted sensitive or irrelevant material
