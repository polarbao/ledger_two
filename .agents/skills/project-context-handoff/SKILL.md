# LedgerTwo Context Handoff

Create a compact continuation brief with:

- Current objective and non-objectives.
- Files changed or investigated.
- Evidence-backed facts and A/B/C/D classification when relevant.
- Current formal docs and current task-card entry, if relevant.
- Decisions made.
- Risks and unresolved questions.
- Next actions and verification status.

Do not include secrets.
