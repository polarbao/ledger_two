# LedgerTwo 项目文档总览

LedgerTwo 是一个双人共享记账 Web 工具。

## 目标

- Mac Air 与 Windows PC 双环境开发
- VSCode + Codex 辅助开发
- 前端 React + TypeScript + Vite
- 后端 Go + SQLite
- Docker Compose 部署到群晖 NAS

## 产品核心

双人共同生活支出的记录、分摊、结算系统，重点回答：本月花了多少、谁付款、谁承担、谁应转给谁。
