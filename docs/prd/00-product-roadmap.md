# 产品定位与版本路线

## 1. 产品定位

LedgerTwo 是一个面向双人、家庭、小范围成员的私有化共享记账系统。核心不是复杂会计，而是解决共同生活支出的记录、分摊、统计和结算。

## 2. 核心问题

1. 谁付款。
2. 谁参与消费。
3. 每个人应承担多少。
4. 当前谁应该转给谁。
5. 哪些账单已经结算。
6. 数据如何长期安全保存。

## 3. 当前版本状态

当前 Demo 已完成本地前后端联调，下一阶段进入 v0.2 稳定真实可用版。

## 4. 版本路线

| 版本 | 目标 | 模块 |
|---|---|---|
| v0.1 Demo | 跑通核心闭环 | 登录、账单、共同支出、结算、统计、本地联调 |
| v0.2 稳定版 | 真实数据可用 | 审计日志、备份恢复、错误码、权限回归、NAS 部署验证 |
| v0.3 体验版 | 提升记账效率 | 复制一笔、快捷记账、账单模板、周期账单 |
| v0.4 数据版 | 减少手工录入 | 微信/支付宝 CSV 导入、导入预览、分类规则、附件 |
| v0.5 家庭版 | 扩展场景 | 多账本、多成员、角色权限、多人结算 |
| v0.6 跨端版 | 移动端长期使用 | PWA、移动端优化、IndexedDB 缓存、离线草稿 |
| v1.0 长期稳定版 | 可维护发布 | CI、测试、迁移、Release、完整备份恢复 |

## 5. 当前优先级

P0：备份恢复、审计日志、权限回归、结算稳定、错误处理。

P1：复制一笔、快捷记账、账单模板、周期账单。

P2：CSV 导入、附件、小票、规则自动化。

P3：多账本、多成员、PWA、离线同步。

## 6. 产品边界

近期不做：股票基金、复杂资产负债、银行自动同步、企业报销审批、复杂复式会计。
