# PRD：统计分析与报表模块

## 1. 模块目标

统计分析模块用于帮助用户理解消费结构、成员承担关系、共同支出趋势和结算情况。

## 2. 当前 Demo 范围

- 月度总支出。
- 月度总收入。
- 分类支出统计。
- 标签统计。
- 成员支出统计。
- 当前待结算金额。

## 3. 后续报表能力

- 月度趋势。
- 年度汇总。
- 分类占比。
- 标签排行。
- 成员实际支付金额。
- 成员实际承担金额。
- 成员净垫付金额。
- 个人消费与共同消费对比。
- 结算历史统计。

## 4. 统计原则

- 消费统计不包含 settlement。
- 删除账单不进入统计。
- 最终金额聚合由后端完成，前端只展示。
- API 金额单位为分。

## 5. 验收标准

- 切换月份后统计刷新。
- 分类统计金额与流水一致。
- 成员统计区分实际支付和实际承担。
- 结算记录不会被计入消费支出。
