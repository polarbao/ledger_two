# PRD：导入、导出、规则自动化模块

## 1. 模块目标

保证数据可迁移、可备份、可导入，并减少长期手动记账成本。

## 2. 导出能力

- CSV 导出：面向表格查看和人工分析。
- JSON 全量导出：面向迁移和恢复。
- SQLite 备份下载：面向完整数据保护。
- 按月份、账本、成员筛选导出。

## 3. 导入能力

后续支持：

- 微信 CSV。
- 支付宝 CSV。
- 银行 CSV。
- 自定义 CSV 模板。

导入流程：上传文件 -> 解析预览 -> 字段映射 -> 去重检查 -> 用户确认 -> 批量写入。

## 4. 规则自动化

示例：

- 商户包含 星巴克 -> 分类餐饮，标签咖啡。
- 商户包含 滴滴 -> 分类交通，标签打车。
- 商户包含 美团 -> 分类餐饮，标签外卖。

## 5. 验收标准

- 用户可以导出 CSV / JSON。
- SQLite 备份可下载。
- 导入预览不直接写入数据库。
- 重复导入不会生成重复账单。
