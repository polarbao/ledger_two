# PRD：附件、小票、OCR 预留模块

## 1. 模块目标

为账单保存消费凭证，例如小票、截图、发票图片等。初期只做附件上传和查看，OCR 放到后续版本。

## 2. 功能范围

v0.4 建议支持：

- 上传图片附件。
- 附件关联到账单。
- 查看附件。
- 删除附件。
- NAS 本地文件存储。

后续可选：

- OCR 识别金额。
- OCR 识别商户。
- OCR 识别日期。
- 自动推荐分类和标签。

## 3. 文件限制

- 支持 jpg、png、webp、pdf，pdf 后期可选。
- 单文件大小建议限制 10MB。
- 每笔账单附件数量建议限制 5 个。

## 4. 验收标准

- 账单详情页可以查看附件。
- 删除账单时附件仍可追踪或进入软删除状态。
- 附件文件不会存入数据库 BLOB。
- 备份说明中明确附件目录需要一起备份。
