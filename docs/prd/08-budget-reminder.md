# PRD：预算、提醒、周期账单模块

## 1. 模块目标

预算与提醒模块用于帮助用户提前控制消费、管理固定周期账单，并提醒长期未结算事项。

## 2. 预算能力

后续支持：

- 月度总预算。
- 分类预算。
- 成员预算。
- 共同支出预算。

## 3. 提醒能力

提醒类型：

- 预算超额提醒。
- 周期账单提醒。
- 长时间未结算提醒。
- 备份失败提醒。

## 4. 周期账单

支持周期：

- 每周。
- 每月。
- 每年。
- 自定义周期。

初期策略：只提醒和手动确认生成，不自动扣账。

## 5. 验收标准

- 可以创建周期账单模板。
- 到期后显示待确认提醒。
- 用户确认后生成真实账单。
- 预算超额时有明确提示。
