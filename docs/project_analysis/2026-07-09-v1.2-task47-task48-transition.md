## 2. Task48 冻结决策

Task48 的正式去重映射采用独立表：

```text
transaction_import_refs
```

决策理由：

- 不直接修改核心 `transactions` 表，降低导入模块对主账单模型的侵入。
- `transaction_import_refs(ledger_id, import_hash)` 作为同账本内最终唯一约束。
- 预览行当前使用扩展后的 `import_items`，其业务语义等同于实施契约中的 `import_rows`。
