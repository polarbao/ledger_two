## 5. Release metric

`auto_correction_rate` uses committed/imported rows produced by a controlled graded acceptance run and matched to a high-confidence, auto, `origin=learned` rule. The numerator is rows later saved as manual/bulk whose final category or order-insensitive complete tag set differs from the preserved suggestion; the denominator is all such committed/imported learned-rule matches. Do not mix suggest-mode samples into this release gate.

1. Sample below 10 is insufficient and keeps `IMPORT_CLASSIFICATION_MODE=suggest`.
2. Rate greater than or equal to 10% blocks learned auto and downgrades it to suggest.
3. Zero samples never authorizes graded production behavior.
