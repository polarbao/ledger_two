# UI：整体布局与导航

## 桌面端

推荐结构：左侧导航栏 + 顶部账本状态区 + 主内容区 + 右侧详情/编辑抽屉。

一级导航：

- 首页 Dashboard。
- 流水 Transactions。
- 结算 Settlement。
- 统计 Analytics。
- 设置 Settings。

## 移动端

推荐结构：顶部账本标题 + 底部 Tab + 悬浮/居中记账入口。

底部 Tab：

- 首页。
- 流水。
- 记账。
- 结算。
- 我的。

## 响应式断点

- mobile：小于 768px。
- tablet：768px 到 1024px。
- desktop：大于 1024px。

## 状态提示

顶部或设置页展示：已同步、同步中、离线、同步失败。
