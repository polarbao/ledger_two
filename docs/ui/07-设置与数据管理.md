# UI：设置与数据管理

## 1. 页面目标

设置页用于管理基础配置、分类、标签、账户、数据导出、备份状态和系统信息。

## 2. 页面分区

### 2.1 个人设置

- 当前登录用户。
- 显示名称。
- 修改密码。
- 退出登录。

### 2.2 分类管理

- 支出分类。
- 收入分类。
- 分类颜色。
- 分类图标。
- 排序。
- 归档。

交互：点击分类进入编辑抽屉，归档前提示该分类不会再出现在新增账单表单中，但历史账单仍保留。

### 2.3 标签管理

- 标签列表。
- 热门标签。
- 新增标签。
- 合并重复标签，后期可选。

### 2.4 账户管理

- 现金。
- 微信。
- 支付宝。
- 银行卡。
- 信用卡。
- 其他。

Demo 阶段账户只作为支付来源，不做复杂余额校准。

### 2.5 数据管理

- 导出 CSV。
- 导出 JSON。
- 下载 SQLite 备份。
- 手动触发备份。
- 查看最近备份记录。

### 2.6 系统状态

- 服务版本。
- 数据库路径。
- 最近备份时间。
- 当前访问方式。
- 同步状态。

## 3. 移动端交互

设置页使用分组列表，每个分组点击进入二级页面。数据导出、备份恢复属于高风险操作，按钮需要二次确认。

## 4. 空状态与异常状态

- 没有备份：显示“暂未生成备份”。
- 备份失败：显示失败原因和重试按钮。
- 导出失败：提示检查 NAS 存储空间和服务日志。

## 5. 验收标准

- 可以查看分类、标签、账户。
- 可以归档分类。
- 可以触发导出和备份。
- 高风险操作都有二次确认。
